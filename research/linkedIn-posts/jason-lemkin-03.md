**Full Text**: We spend 5x more on the AI agents that run on top of our CRM than on our CRM itself.

Our Salesforce contract is real money. But our agent stack … Artisan (AI SDR), Qualified (inbound BDR), Agentforce, Momentum, Monaco, plus a bit to run Qbee and 10K .. that’s where the serious budget goes. 

The CRM is now infrastructure. The agents are the product.

Side note: we’re basically ending use of Marketo. For years it functioned almost like a secondary CRM. But its API is so dated our agents can’t work with it. If your agents can’t read from it and write to it cleanly, it doesn’t matter how much history lives inside it. It becomes a data silo.

A lot of tools that were mission-critical five years ago are going to get quietly deprecated over the next 18 months. Not because they got worse, but because they never updated their APIs for an agent first world.

Agent compatibility is the new integration checklist.

**Pattern**: Current observation → Economic inversion → Specific examples → Technical reason → Future prediction
**Key Insight**: The claim isn’t:
AI tools are expensive.
The real claim is:
Value is moving up the stack.
Before:
Infrastructure → CRM → Workflow → Human
Now:
Infrastructure → CRM → Agent → Outcome
CRM becomes storage and system-of-record.
Agents become the active layer that:
reads data
decides
acts
updates systems
The money follows whoever produces outcomes.

**My Note**: Technology rarely disappears because it stops working.

It disappears because the surrounding ecosystem changes.

The strongest products don’t just survive platform shifts.

They become the new infrastructure.
In an agent-first world, software value shifts from storing data to acting on data.
**Repurpose Angle**: APIs used to be technical debt.

Now APIs become distribution.
